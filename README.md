# device-list


## Step 1: Create a file with name .env.local and paste inside:
```
NODE_ENV=development
```

## Step 2: Project setup
```
yarn install
```

## Step 3: Compiles for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
